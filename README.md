# Enterprise-Network-PT-Lab
I made this to cement most of the stuff I learned for the CCNA in my memory. It's mostly functional.

A work in progress, there's still a number of things I need to make work, like the phones. For that I still need to add a DHCP server and then configure DHCP helper addresses.
I started to setup a TFTP server, so I need to finish that as well.
Right now the ASBR1 router is connecting to the external router over a serial connection.....need to get some fiber installed for that lol.

At some point I'll probably end up putting some actual core switches in this lab just so that I can get a better idea of how the core switches at work are built and connected.
For the time being I think this covers all the fundamentals though, dynamic and static routing, switching, intervlan communication, port channels, etc. If you boot up the lab now, any PC on either side of the routers will be able to ping each other.
